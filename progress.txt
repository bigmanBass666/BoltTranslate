# TranslateSharp 开发进度

## 2026-04-17

### 已完成

- **UI 美化**：悬浮窗重新设计（圆角、边距、关闭按钮）
- **失焦关闭修复**：移除失焦自动关闭，只保留 X 和 ESC
- **托盘重启功能**：右键菜单添加"重启"选项
- **FreeConsole**：启动时释放控制台窗口
- **Config 首次配置提示**：未配置 API Key 时自动打开记事本并等待编辑完成
- **剪贴板污染（架构级修复）**：
  - 改为单个 STA 线程完成所有剪贴板操作，避免多线程竞争
  - 空选中检测：`selectedText == savedText` 时返回空
  - 恢复时区分空字符串（Clear）和非空（SetText），带重试机制
  - GetClipboardSequenceNumber 序列号检测
- **UI Automation（实验性）**：COM IUIAutomation 原生接口获取选中文字，不碰剪贴板
- **去重逻辑**：缓存 `_lastTranslatedText`，避免重复翻译
- **悬浮窗定位**：全部 Win32 API（GetCursorPos/SetWindowPos/GetSystemMetrics），光标坐标兜底
- **HttpClient 资源管理优化**：
  - TranslationService 实现 IDisposable 接口
  - Configure 方法中释放旧 HttpClient 实例后再创建新实例
  - App.OnExit 中调用 TranslationService.Dispose() 确保资源正确释放

## 2026-04-19

### 已完成

- **悬浮窗定位 — 固定偏移方案（最终方案）**：
  - 经过 7 次动态定位尝试全部失败（DPI 缩放、纯 WPF、SetWindowPos 覆盖、全物理坐标等）
  - 改为光标右下方固定偏移 +20/+20 物理像素
  - 删除 CalculatePosition 等复杂坐标计算，仅保留基本边界检测
  - SelectionService 使用 GetPhysicalCursorPos 获取物理坐标
  - 编译 0 错误 0 警告
- **常量类提取（代码质量优化）**：
  - 创建 Services/AppConstants.cs 统一管理所有魔法数字和硬编码字符串
  - 包含：弹窗定位参数、热键冷却时间、剪贴板操作延迟/超时、翻译 API 参数和提示词
  - 更新 WindowManager.cs：移除本地常量，使用 AppConstants
  - 更新 HotkeyService.cs：热键冷却时间引用 AppConstants
  - 更新 ClipboardService.cs：所有延迟、轮询次数、超时时间引用 AppConstants
  - 更新 TranslationService.cs：系统提示词、温度、最大 token、超时时间引用 AppConstants
  - 编译 0 错误，3 个原有警告（与本次修改无关）
- **Win32 API 集中管理**：
  - 创建 Services/NativeInterop/Win32Api.cs 集中所有 P/Invoke 声明
  - 包含：POINT 结构体、窗口定位/样式常量、热键/键盘常量、12 个 Win32 方法
  - 命名空间：TranslateSharp.Services.NativeInterop
- **HotkeyService 独立服务**：
  - 从 SelectionService 提取热键管理逻辑，创建独立的 HotkeyService
  - 接口 IHotkeyService：RegisterHotkey/Start/Stop
  - 使用 Win32Api 集中管理的 P/Invoke，不再自声明
  - 冷却时间 500ms，异常静默忽略（不弹 MessageBox）
  - 选中文本获取：先调用 UiaSelectionService.TryGetSelectedText()，空则返回空字符串
- **TextSelectionService 重构**：
  - 将静态类 UiaSelectionService 重构为 ITextSelectionService 接口 + TextSelectionService 实例类
  - 通过依赖注入接收 IClipboardService，协调 UIA 和 Clipboard 两种选中文本获取方式
  - UIA 失败时静默返回 null，自动降级到 Clipboard 方式
  - 更新 HotkeyService 和 SelectionService 的构造函数，注入 TextSelectionService
  - 简化 SelectionService：删除重复的 Clipboard 操作代码和 Win32 API 声明
  - 删除旧的 UiaSelectionService.cs 文件
  - 编译 0 错误 0 警告
