# TranslateSharp 开发进度

## 2026-04-17

### 已完成

- **UI 美化**：悬浮窗重新设计（圆角、边距、关闭按钮）
- **失焦关闭修复**：移除失焦自动关闭，只保留 X 和 ESC
- **托盘重启功能**：右键菜单添加"重启"选项
- **FreeConsole**：启动时释放控制台窗口
- **Config 首次配置提示**：未配置 API Key 时自动打开记事本并等待编辑完成
- **剪贴板污染（架构级修复）**：
  - 改为单个 STA 线程完成所有剪贴板操作，避免多线程竞争
  - 空选中检测：`selectedText == savedText` 时返回空
  - 恢复时区分空字符串（Clear）和非空（SetText），带重试机制
  - GetClipboardSequenceNumber 序列号检测
- **UI Automation（实验性）**：COM IUIAutomation 原生接口获取选中文字，不碰剪贴板
- **去重逻辑**：缓存 `_lastTranslatedText`，避免重复翻译
- **悬浮窗定位**：全部 Win32 API（GetCursorPos/SetWindowPos/GetSystemMetrics），光标坐标兜底
- **HttpClient 资源管理优化**：
  - TranslationService 实现 IDisposable 接口
  - Configure 方法中释放旧 HttpClient 实例后再创建新实例
  - App.OnExit 中调用 TranslationService.Dispose() 确保资源正确释放
