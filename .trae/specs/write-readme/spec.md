# 使用文档 README Spec

## Why
项目缺少面向最终用户的使用说明文档，需要编写 README 教用户如何安装、配置和使用 TranslateSharp。

## What Changes
- 新增 `README.md` 文件（中文）

## Impact
- 新增文件: `README.md`

---

## ADDED Requirements

### Requirement: 使用文档
系统 SHALL 提供一份完整的中文使用文档，包含以下内容：

#### 内容要求
1. **项目简介**：一句话描述 TranslateSharp 是什么
2. **功能特性**：列出核心功能
3. **截图/演示**：（可选）如果有
4. **环境要求**：.NET 8 / Windows 10+ 
5. **安装方式**：
   - 从源码编译运行
   - 或下载发布版 exe
6. **配置说明**：
   - config.json 各字段含义
   - API Key 配置步骤
7. **使用方法**：
   - 快捷键操作
   - 托盘图标功能
8. **常见问题**：
   - 热键被占用怎么办
   - 翻译不准确怎么办
